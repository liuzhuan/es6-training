# 字符串新方法：`padStart` 和 `padEnd`

分别用于在字符串前后填充字符，直到达到指定长度。

```javascript
'x'.padStart(5, 'ab') // 'ababx'
'x'.padEnd(5, 'ab') // 'xabab'
```

So easy!
